Country Explorer Web App

Project Overview:

This project is a web application that allows users to explore country data using a public API. It demonstrates API integration, dynamic UI rendering, and the use of JavaScript higher-order functions.

------------------------

Features:

* 🔍 Search countries by name
* 🌏 Filter countries by region (Asia, Europe, etc.)
* 📊 Sort countries by population
* 🌙 Dark mode toggle
* 📱 Responsive layout

------------------------

Technologies Used

* HTML
* CSS
* JavaScript
* Fetch API

------------------------

API Used:

REST Countries API
https://restcountries.com/

Data is fetched using:
https://restcountries.com/v3.1/all?fields=name,flags,population,region

------------------------

Concepts Used

* Fetch API for data retrieval
* DOM manipulation
* Array Higher-Order Functions:

  * map()
  * filter()
  * sort()
* Event handling

------------------------

How to Run the Project Locally

1) Download or clone the repository
2) Open the project folder in VS Code
3) Install the Live Server extension (if not already installed)
4) Right-click on 'index.html'
5) Click "Open with Live Server"
